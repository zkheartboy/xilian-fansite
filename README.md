# 昔涟应援网站

> 来自星尘角色昔涟的粉丝网站

## 🏠 在线预览

网站部署后访问：`https://xilian-fans.github.io/xilian-fansite/`

## 🚀 快速开始

### 安装依赖

```bash
npm install
```

### 开发模式

```bash
npm run dev
```

访问 http://localhost:4321

### 构建

```bash
npm run build
```

### 部署

1. 推送代码到 GitHub
2. 启用 GitHub Pages（Settings → Pages → GitHub Actions）
3. 自动部署

## 📁 项目结构

```
xilian-fansite/
├── src/
│   ├── components/    # 组件
│   ├── content/       # 内容（新闻等）
│   ├── data/         # 数据配置
│   ├── layouts/      # 页面布局
│   └── pages/        # 页面
├── public/           # 静态资源
├── astro.config.mjs # Astro 配置
└── tailwind.config.mjs
```

## 📝 页面

- `/` - 首页
- `/about` - 角色介绍
- `/gallery` - 图片库
- `/videos` - 视频库
- `/news` - 资讯
- `/community` - 社区
